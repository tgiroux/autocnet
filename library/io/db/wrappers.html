io.db.wrappers — Database Wrappers to Data Structures¶

The io.db.wrappers module provides wrappers for Pandas DataFrames and Series objects.

New in version 0.1.0.

class autocnet.io.db.wrappers.DbDataFrame(*args, **kwargs)[source]¶

Lightweight subclass of a pandas dataframe that overrides __setitem__ to get changes pushed back into a database. The class functions by calling the parent’s (pd.DataFrame) __setitem__ (thereby using all of the Pandas logic) and then calling whatever setter might be defined.
